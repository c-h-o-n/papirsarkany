# papirsarkany

This is official the repositoy for the [papisarkany.hu](https://papirsarkany.hu) webshop.

# Prerequisites

- node 22
- pnpm 9

## Getting Started

First, run the development server:

```bash
pnpm i
pnpm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
